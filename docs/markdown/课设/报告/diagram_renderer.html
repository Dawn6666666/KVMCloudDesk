截图说明

1. 推荐选择“白底纯净 (截图)”以输出标准的报告无背景插图。

2. 调整浏览器缩放到 100%。

3. 按系统快捷键截图：
- Win: Win + Shift + S
- Mac: Cmd + Shift + 4

4. 截图完成后，按键盘 ESC 键立刻复原侧栏与导航。

提示：当前图表比例已专门为 Word 及 LaTeX 论文级排版调优。支持一键隐藏所有界面面板以获得纯净插图。
